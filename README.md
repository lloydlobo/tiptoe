## Controls

### Keyboard

- WASD and Arrow keys: Move in direction (Up Left Down Right)
- SPACE: Float up
- X-`<direction>`: Dash
- ESCAPE:
- Access menu
- Return to previous menu
- Toggle gameplay/menu
- ENTER: Confirm actions
